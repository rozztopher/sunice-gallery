# README

## Setup

Download [Node.js](https://nodejs.org/en/download/) and run the following commands:

```bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
